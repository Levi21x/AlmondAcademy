## 2. The concept (the idea behind the mark)

**Lead direction — "The Knowledge Seed" (RECOMMENDED):**
A clean, geometric **almond seed** silhouette (pointed apex, rounded base). Inside it — formed from
negative space and one continuous hairline — sits a single luminous **neural node with 2–3 fine dendrites**
(intelligence / "brain food"), lit from within like a seed of knowledge. Warm amber gradient on deep charcoal.
The pointed apex naturally reads as an **"A"**, tying the symbol to the name.

**Alternate cues** (pick ONE inside the almond — never combine, keep it minimal):
- a single neural node + thin dendrites (intelligence) ← default
- a delicate upward sprout / spark (knowledge growing)
- a soft brain-fold gyrus curve (the mind)

**Monogram fallback** — "A-seed": the letter **A** whose triangular counter (inner hole) is shaped like an almond seed.
Extremely scalable; good if the pictorial mark gets muddy at 16px.

> Keep it to ONE idea. No stethoscopes, no medical crosses, no sparkles/glitter, no orbiting rings — those are clichés.

## 3. Exact brand tokens (give these to the model)

**Colors**
- Charcoal base `#131313` (deepest `#0d0d0d`), surface `#1c1b1b` / `#1f1f1f`
- Almond / amber accent `#d5c5a8` → amber-light `#e6d5b8` → warm cream `#fff2de` (the mark's gradient)
- Coral warmth (optional secondary) `#e4b4a0`
- Hairline / borders `#353534`

**Type**
- Wordmark: **Bricolage Grotesque**, weight 800, tight tracking (`-0.03em`)
- Casing: **"Almond"** in warm cream `#fff2de` + **"AI"** in amber `#d5c5a8` (tight, one word, no space)

### 4d. Useful variations (run as needed)

- **Rounded app tile (store/favicon):** take 4a and append —
  `Place the mark on a rounded-square (squircle) tile with a subtle charcoal-to-near-black radial gradient
  and a 1px inner amber hairline. iOS app-icon style, soft depth.`
- **Monogram lockup:** swap the symbol description for —
  `the letter "A" whose triangular inner counter is shaped like an almond seed, amber gradient on charcoal,
  geometric and ultra-scalable.`
- **Light-mode / print version:** replace background with `warm bone #f7f6f3` and make the almond a solid
  warm espresso-amber `#9a7c4f` so it holds contrast on light.
- **Monochrome / single-color:** `single flat color, no gradient — the almond and synapse in one warm amber
  #d5c5a8 on charcoal (and an inverted charcoal-on-cream version), for stamps, embroidery, and favicons.`

## 6. Tips for getting a clean result

- **Generate the symbol (4a) first**, lock the one you love, then ask the model to "use this exact mark"
  for the lockup (4b) and board (4c) so the symbol stays consistent across all three artifacts.
- Gemini/Imagen/Ideogram: paste the prose prompt as-is. **Ideogram** is best if you also want the
  "AlmondAI" wordmark rendered with correct, crisp lettering.
- **Midjourney:** append `--style raw --v 6 --ar 1:1` (icon) or `--ar 16:9` (board); convert the prose to
  the same description — it still works well.
- Ask for **"flat vector, solid background, centered, lots of padding"** to get something you can actually
  trace/clean up in Figma or Illustrator afterward.
- Generate **4–8 variations** and pick; logos are a numbers game. Test the winner at **16×16 px** — if the
  inner synapse disappears, fall back to the monogram (4d) for the favicon.
- Keep the palette locked across every generation so icon, logo, and board read as one brand.
